    • Game Development Resources
    Interested in game development? Try checking the following game development technology below!

    Unity
    Unity is a free to use 3D development platform that doubles as a popular game engine. Unity offers a variety of tools including: a full 3D and 2D physics renderer, a flexible scripting to allow for various functionalities; as well as, a diverse asset store for both 3D and 2D games. Unity’s main scripting language is C# and includes many easy to use functions.
  • Unity Website
  • Unity Documentation
  • Tutorial
  • Various Assets

    • GoDot
    Godot is a free, open-source game engine under the permissive MIT license. Godot offers a huge set of common tools such as: Nodes for all your needs, Flexible scene system, Visual editor, Friendly content creation pipeline, Persistent live editing, and the ability to create your own custom tools. This game-engine supports a variety of languages (C# 7.0, C++, python, Nim, etc.) as well as python-like scripting language called GDScript to create both 2D and 3D games.
  • Go Dot Website
  • Go Dot Documentation
  • Go Dot Tutorial

    • PyGame
    Those at UCI might be very familiar with this simple, game creating 3rd party python library. PyGame is a free, open source library that allows for an easy way to create 2D games through the use of Python.
  • PyGame Getting Started
  • PyGame Tutorial
